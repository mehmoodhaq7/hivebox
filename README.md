# HiveBox 🐝

A scalable RESTful API for beekeepers using openSenseMap data.

## Version

Current version: v0.0.1

## What is HiveBox?

HiveBox fetches temperature data from beehive-mounted sensors
via openSenseMap API and exposes it through a REST API.
